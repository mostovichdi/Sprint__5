# Sprint__5
Проект содержит 6 модулей с тестированием:
- test_registration - Тесты по тестированию регистрации нового пользователя
- test_login - Тесты входа в личный кабинет
- test_logout - Тесты выхоа из личного кабинета
- test_go_to_account_profile - Тесты перехода в личный кабинет
- test_from_account_to_constructor - Тесты перехода из личного кабинета в конструктор
- test_section_constructor -Тесты переключения по разделам в конструкторе

Дополнительные модули:
- data - Содержит данные для входа в личный кабинет, и ожидаемые тексты для проверок
- generation_login - Генерация логина и пароля для проверки регистрации
- locators - Локаторы, для поиска элементов на странице
- urls - сслыки на страницы
